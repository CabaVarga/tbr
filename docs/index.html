What it does

tbr counts your open tabs, compares that count to your warning and danger thresholds, and escalates the response as the number climbs.

  • Popup first Check the count and tune the thresholds from one place.
  • Visible warnings Badge is the default cue; page border and icon changes are optional.
  • Dedicated interruption The warning window interrupts the habit once the danger line is crossed.

Permission rationale

tabs

Read the current tab count and react to tab create, close, activate, replace, and update events.

storage

Save warning thresholds and visual preferences locally in your browser.

scripting

Inject and remove the optional warning border on the active HTTP or HTTPS page.

<all_urls>

Allow the warning border to work on whichever page is active when it is enabled.

tbr is a local browser extension with a public support page and privacy policy.
